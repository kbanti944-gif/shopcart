📋 How It Works

Shopkeeper Steps:

  1. 1. Register - Shop name, phone, address
  2. 2. Add Products - Name, price, quantity
  3. 3. Share Link - Send catalog via WhatsApp
  4. 4. Get Orders - Customers send carts
  5. 5. Deliver - Prepare & hand over items

Consumer Steps:

  1. 1. Open Link - From shopkeeper's WhatsApp
  2. 2. Browse - See all products & prices
  3. 3. Select - Choose items & quantity
  4. 4. Create Cart - Generate order summary
  5. 5. Send & Get - Share cart, receive items
